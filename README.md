# Java 面向对象程序设计 - 课程实验

> 🏫 **华中师范大学 (CCNU) · Java 课程作业**
>
> 👨‍🏫 授课教师：张连发

---

## 📖 简介

本项目为 Java 面向对象程序设计课程的实验代码合集，涵盖了从基础语法到高级特性的完整学习路径。所有实验均使用 IntelliJ IDEA 开发，基于 JDK 标准库实现。

---

## 🗂️ 实验目录

| 实验 | 内容 | 知识点 |
|------|------|--------|
| **lab01** | Java 入门 | HelloWorld、基本程序结构 |
| **lab02** | 数据类型与控制语句 | 基本类型、循环、数组、条件判断、数学运算 |
| **lab03** | 类与对象 | 构造方法、封装、方法重写（equals/toString）、复合类设计 |
| **lab04** | 继承与多态 | 继承、变量隐藏、抽象类、接口、向上转型 |
| **lab05** | 内部类与异常处理 | 成员内部类、匿名内部类、自定义异常、接口回调 |
| **lab06** | 字符串与正则表达式 | Character 类、字符串反转/回文检测、正则匹配、大整数运算 |
| **lab07** | 输入/输出流 | 文件字节流复制、缓冲流性能对比、文本统计、序列化 |

---

## 🛠️ 技术栈

- **语言**：Java (JDK 17+)
- **开发工具**：IntelliJ IDEA
- **构建工具**：无（标准 Java 项目结构）

---

## 🚀 运行方式

1. 用 IntelliJ IDEA 打开项目根目录
2. 在 `src/` 目录下找到对应实验的 `Main` 类
3. 右键 → Run 即可运行
